### 结构说明

- Baseline 探究了 yolo 和 rtdetr 两种模型的可行性，证明了 yolo 系列效果更好；
- v1 是修改为架构的版本。由于训练结果太多全删掉了，最好的 pt 文件之前发在群里面了。修改完架构训练结果没有变好，可能是由于数据量太少造成过拟合。🥵
- v2 是将输入图片从 1 通道转换为 3 通道的版本。由于转换后的数据集超过 100MB 了，上传不了，运行 preprocess_image.py 和 augmenty_yolo_dataset.py 可以得到增强后的数据集，但是用增强后的数据集跑完结果也是一般。这个版本还使用了融合后的数据，发现增加数据对提示模型效果有很大帮助。目前测试下来使用 1：1 融合数据与原始数据+v1 版本中的模型在只训练 50 轮的情况下修改能够到达` mAP50=95%`的性能，且损失函数并未完全收敛。🤩
