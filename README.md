### 结构说明

- Baseline 探究了 yolo 和 rtdetr 两种模型的可行性，证明了 yolo 系列效果更好；
- v1 是修改为架构的版本。由于训练结果太多全删掉了，最好的 pt 文件之前发在群里面了。修改完架构训练结果没有变好，可能是由于数据量太少造成过拟合。🥵
- v2 是将输入图片从 1 通道转换为 3 通道的版本。由于转换后的数据集超过 100MB 了，上传不了，运行 preprocess_image.py 和 augmenty_yolo_dataset.py 可以得到增强后的数据集，但是用增强后的数据集跑完结果也是一般。这个版本还使用了融合后的数据，发现增加数据对模型性能提升没有什么作用，可能是生成后数据仍然很少，改变不了小数据集的本质，转换通道也没什么效果。😢
- v3 和 v3_test 尝试将将YOLO和DINO架构结合起来，效果依然一般。
- v4 版本尝试将DINOv3蒸馏到YOLO11n的骨干网络上。
